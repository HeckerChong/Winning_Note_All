三、UPDATE 命令

update table_name set 列=表达式 
[FROM table_sources] 
[WHERE search_condition]

1、使用简单的update

UPDATE YF_YFZKC SET djsl = 0

2、 把WHERE子句和UPDATE语句一起使用

UPDATE YF_YFZKC SET jxje=0 Where abs(jxje)>1000000

3、通过UPDATE语句使用来自另一个表的信息

update #temp set ksmc =b.name 
from #temp a,YY_KSBMK b where a.ksdm=b.id

4、在游标的当前行上使用update

update #dxmdm set zje=@zje where CURRENT OF cs_dxmdm
